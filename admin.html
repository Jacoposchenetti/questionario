📊 Configura Google Sheets

Per esportare direttamente in Google Sheets serve un OAuth 2.0 Client ID di Google. Nella GCP Console:
1. Abilita l'API Google Sheets
2. Crea credenziali → ID client OAuth 2.0 → Applicazione web
3. Aggiungi https://jacoposchenetti.github.io come origine JavaScript autorizzata
4. Copia il Client ID qui sotto.
